# Relaxess — Official Website

Production-ready static website for Relaxess wellness application.

## Contents

- `index.html` - Main website entry point
- `assets/` - CSS, JavaScript, and images
  - `images/` - All website images (logo, hero, mockups)

## Deployment

### Cloudflare Pages
1. Extract the ZIP
2. Upload contents to Cloudflare Pages
3. Configure custom domain in Cloudflare settings

### GitHub Pages
1. Extract the ZIP
2. Push to GitHub repository
3. Enable GitHub Pages in repository settings

## Features

- Dark and Light themes
- 6 language support (English, Spanish, German, French, Portuguese, Japanese)
- Responsive design (mobile, tablet, desktop)
- Premium glassmorphism design
- Smooth animations and transitions
- All images included locally

## Pages

- Home - Hero section with app preview
- Features - Relaxess capabilities
- Relaxation Tools - Guided exercises and soundscapes
- Your Companion - AI companion information
- Pricing - Subscription options
- Support - FAQ and contact information
- Privacy Policy
- Terms of Use
- Contact

## Technical Stack

- React 19
- Tailwind CSS 4
- Framer Motion (animations)
- Wouter (routing)
- shadcn/ui (components)
