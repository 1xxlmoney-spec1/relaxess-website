APP NAME:
CalmSpace – AI Companion for Relaxation

---

PROJECT OVERVIEW:

Create a scalable mobile application called “CalmSpace”.

“CalmSpace” is an ultra-minimalist AI-powered application designed for emotional support and relaxation. It is intended for people experiencing stress, anxiety, overthinking, emotional exhaustion, or sleep difficulties.

The application combines:

* AI emotional support chat
* Calming soundscapes
* Minimalist UX design
* An emotionally relaxing experience

The application must be designed with future scalability in mind, allowing new features, audio content, languages, and updates to be added easily without rebuilding the entire system. The architecture should also support future advertising integration.

This is NOT a medical or therapy application.
It does NOT diagnose or treat mental health conditions.

---

SUPPORTED LANGUAGES (MVP VERSION):

At launch, the application must support 3 languages:

1. English
2. Spanish
3. German

English should be the default language on first launch.

The architecture must allow easy future expansion for:

* Japanese
* French
* Portuguese
* and additional languages

The language management system must be modular and scalable.

---

CORE PRODUCT PRINCIPLE:

The app should feel like:
“A calm emotional space inside the phone.”

Users should feel calmer and more relaxed within the first 30–60 seconds after opening the app.

Avoid complexity.
Avoid information overload.
Avoid aggressive visual elements.

The UX design should help emotionally overwhelmed users instantly feel safe and relaxed.

---

UI / UX DESIGN RULES:

* Minimalist Apple-style interface
* Light and dark screen themes:

  * Light theme: standard white interface
  * Dark theme: based on the provided image file “Dark” (default theme). When the AI chat is activated, the chat background should smoothly transition into a pixelated version of the same image using the JPG file “Dark Pix”. Both “Dark” and “Dark Pix” are attached files.
* Maximum simplicity
* Minimal number of buttons (beautiful and clear)
* Calm and soft visual style
* No visual clutter
* No complicated onboarding process

Users should be able to access the AI chat within 1–2 taps maximum.

---

THEME SYSTEM:

The application must support BOTH modes:

1. Dark Mode (default)
2. Light Mode

Users must be able to manually switch themes in settings.

DETAILED RELAXATION TOOLS LOGIC:

1. Breathing Exercise (4-4-6)
   (Available in both free and premium versions)

Purpose:
Help users quickly calm anxiety and reduce emotional stress.

User flow:

* User taps “Breathing Exercise”
* A calm breathing screen opens
* A softly animated breathing circle appears

Breathing cycle:

* Slow inhale for 4 seconds
* Hold breath for 4 seconds
* Slow exhale for 6 seconds

Visual guidance:

* Circle slowly expands during inhale
* Circle pauses during breath hold
* Circle slowly contracts during exhale

Example text prompts:

* “Slowly breathe in...”
* “Gently hold your breath...”
* “Slowly breathe out...”

Background:

* calming music continues playing
* minimalist interface
* no distracting elements

Recommended session duration:
At least 30–60 seconds.

---

2. Sleep Mode
   (Premium only)

Purpose:
Help users emotionally relax before sleep.

Flow:

* User taps “Sleep Mode”
* Interface becomes dimmer and softer
* AI responses become slower and calmer
* Relaxing audio starts automatically

AI communication tone:

* extremely gentle
* emotionally calming
* short responses only

Example phrases:

* “You don’t need to force sleep.”
* “Just allow yourself to rest.”
* “It’s okay to slow down.”

Visual style:

* reduced brightness
* smooth animations
* no stimulating visual elements

---

3. Grounding Exercise
   (Premium only)

Purpose:
Help users manage intrusive thoughts and emotional overload.

Flow:

* User taps “Grounding Exercise”
* AI guides the user through a simplified 5-senses grounding technique

Sequence:

* Name 5 things you can see
* Name 4 things you can touch
* Name 3 things you can hear
* Name 2 things you can smell
* Name 1 emotion you currently feel

AI tone:

* calm
* patient
* non-medical
* supportive

This feature should feel simple and emotionally safe.

---

4. Quiet Relaxation Mode
   (Premium only)

Purpose:
Allow emotionally exhausted users to relax without needing conversation.

Flow:

* User taps “Quiet Mode”
* AI chat becomes inactive
* Only calming audio and relaxing visuals remain active

Features:

* looped relaxing music
* rain or forest sounds
* smooth calming background animation

No notifications.
No chat prompts.
No interruptions.

Users should feel safe, calm, and emotionally protected.
